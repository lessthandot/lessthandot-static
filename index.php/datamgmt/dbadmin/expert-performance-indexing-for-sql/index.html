Expert Performance Indexing for SQL Server 2012 for $15

by Ted Krueger (onpnt) on November 20, 2012 in category Database Administration Microsoft SQL Server Microsoft SQL Server Admin. Article views: 10,501
Tweet Instapaper

I’ve been fortunate to have worked on a few books with some brilliant SQL Server Pros in the past few years.  One of the most recent books with Jason Strate, Expert Performance Indexing for SQL Server 2012, was published by APRESS.  I caught something last week that APRESS is having as a deal for their cyber Monday specials.  Basically, APRESS has all of their books in the alpha stage and eBooks on sale for only $15.  As most everyone knows, I’m a big fan of download free publications.  This book however is not available that way but the $15 is pretty close to that.  Having worked on many parts of the book and reading it a few times, with confidence, I can say it is well worth that.

I’d highly recommend taking advantage of that deal and grabbing an eBook copy of the book.  The coolest part of this book; it really is about the only true indexing book out there.  The book has a deep dive into indexing over other books that cover some index portions as part of optimization and tuning topics.  I’ve already referred back to the book as a reference in my own day-to-day needs.
